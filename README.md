## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation & Running

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Visit `http://localhost:3000` to see the app.

### Production Build

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```
